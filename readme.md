<h3>rough overview:</h3>

The database is controlled with commands separated by hyphens.
- **Commands:**
    - **"cdb"** - create database
    - **"ddb"** - delete database
    - **"cl"**  - create table
    - **"dl"**  - delete table
    - **"i"**   - insert key and value
    - **"g"**   - get value
    - **"rk"**  - remove key
